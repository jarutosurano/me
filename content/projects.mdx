**[byronbaycandles.com](https://byronbaycandles.com)** — *In Progress*

This one's fun. The site was running on a headless WooCommerce stack (Next.js/React/Typesense) and I'm converting it to Blocksy. Solo dev and project manager, using ClickUp and Claude Code.

**What I'm building:**
- Automated Figma design values → Blocksy Customizer workflow (speed is the game)
- Reusable child theme framework for future projects
- Custom search plugin using Typesense, built with Claude Code
