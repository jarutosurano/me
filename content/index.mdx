## What I Actually Do All Day

**WordPress Development** — End-to-end builds using Blocksy and WordPress Full Site Editing (FSE). Figma design goes in, a live website comes out. Theme customization, WooCommerce setup, payment gateways, performance tuning — the whole thing.

**Monthly Site Maintenance** — I keep 30+ websites alive, updated, and not hacked. Core/theme/plugin updates, security monitoring, malware cleanup, and the occasional 2am "the site is down" panic.

**Platform Migrations** — I've moved 100+ websites across 3 major hosting platforms without losing my mind (mostly). Drupal, BigCommerce, headless WooCommerce — if it exists, I've probably migrated it to modern WordPress.

**AI-Augmented Workflow** — Claude Code is my daily co-pilot. SSH scripts for maintenance automation, AI-assisted development, and I'm building toward automated visual regression testing with Playwright. Because manually clicking through 30 sites every month is no way to live.
